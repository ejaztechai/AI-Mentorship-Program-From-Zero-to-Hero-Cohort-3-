## üéØ About the Program

The AI From Zero to Hero Mentorship Program is designed to take you from a beginner with no experience in AI to mastering advanced concepts in Artificial Intelligence and Machine Learning.

This cohort follows a **20-week intensive schedule** divided into three distinct learning phases, culminating in a final project. Each phase is designed to build upon the last, ensuring a strong, sequential grasp of AI principles.

## üìö Core Courses

Below is a list of recommended courses to build your AI knowledge step by step. Enroll in these courses sequentially to gain foundational and advanced AI skills:

### **Phase 1: Foundations (4 Weeks)**
**(Dec 8, 2025 - Jan 4, 2026)**

#### 1. Python Basics
Learn the fundamentals of Python, the most widely used language in AI and Machine Learning.
* [Watch the Python Basics Course]
* **Table of Content:**
    > 0:00:00 Introduction, 0:00:30 What You Can Do With Python, 0:01:15 Your First Python Program, 0:05:30 Variables, 0:09:08 Receiving Input, 0:10:48 Type Conversion, 0:18:49 Strings, 0:23:41 Arithmetic Operators, 0:25:59 Operator Precedence, 0:27:11 Comparison Operators, 0:28:52 Logical Operators, 0:31:06 If Statements, 0:36:16 Exercise, 0:41:42 While Loops, 0:45:11 Lists, 0:48:47 List Methods, 0:52:16 For Loops, 0:54:54 The range() Function, 0:57:43 Tuples

#### 2. AI for Everyone
A beginner-friendly course introducing the concepts of Artificial Intelligence, its applications, and its impact on society.
* [Enroll in AI for Everyone]
* **Weekly Topics:**
    * **Week 1:** What is AI (Machine Learning, data, terminology, deep learning intuition).
    * **Week 2:** Building AI Projects (Workflow of ML/Data Science projects, choosing a project).
    * **Week 3:** AI in Your Company (Case studies, AI team roles, AI Transformation Playbook, pitfalls).
    * **Week 4:** AI and Society (Realistic view of AI, discrimination/bias, adversarial attacks, AI and jobs).

### **Phase 2: Supervised ML (5 Weeks)**
**(Jan 5, 2026 - Feb 8, 2026)**

#### 3. Supervised Machine Learning: Regression and Classification
Dive into the core concepts of supervised learning, including linear regression, logistic regression, and techniques like regularization.
* [Enroll in the Course]
* **Weekly Topics:**
    * **Week 1:** Introduction to Machine Learning (Supervised/unsupervised, Linear regression, Cost function, Gradient Descent).
    * **Week 2:** Regression with Multiple Input Variables (Multiple features, Vectorization, Feature scaling, Feature engineering, Polynomial regression).
    * **Week 3:** Classification (Motivations, Logistic regression, Decision boundary, Cost function, Gradient Descent Implementation).
    * **Topics continued:** The problem of overfitting, Addressing overfitting (Regularization).

### **Phase 3: Advanced Deep Learning (6 Weeks)**
**(Feb 9, 2026 - Mar 22, 2026)**

#### 4. Advanced Learning Algorithms
Master advanced topics like neural networks, decision trees, and hyperparameter tuning.
* [Enroll in the Course]
* **Weekly Topics:**
    * **Week 1:** Neural Network (Neurons, Layers, Forward propagation, TensorFlow implementation, Matrix multiplication).
    * **Week 2:** Neural Network Training (Training Details, Activation functions, Multiclass Classification, Softmax, Advanced Optimization).
    * **Week 3:** Advice for Applying Machine Learning (Model Evaluation, Diagnosing bias/variance, Regularization, Learning curves, Error analysis, Transfer learning).
    * **Week 4:** Decision Tree (Decision tree model, Learning Process, Information Gain, Continuous valued features, Regression Trees).
    * **Topics continued:** Using multiple decision trees, Random forest algorithm, XGBoost.
